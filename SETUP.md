# Nuclora Website тАФ Setup Guide

## Quick Start

1. Open a terminal in this folder
2. Install dependencies:
   ```
   npm install
   ```
3. Run the development server:
   ```
   npm run dev
   ```
4. Open http://localhost:3000 in your browser

## Build for Production
```
npm run build
npm start
```
